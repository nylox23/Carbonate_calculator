# Carbonate_calculator

## Репозиторий для итогового проекта курса «Разработка Интернет Приложений (РИП)» 2025

### Содержит окончательную версию веб-приложения для расчета объема газа в результате реакции карбоната кальция с кислотами. Также включены все дополнительные задания, полученные по ходу курса.
