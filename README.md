# Carbonate_calculator

## Репозиторий для итогового проекта курса «Разработка Интернет Приложений (РИП)» 2025

### Содержит окончательную версию веб-приложения для расчета объема газа в результате реакции карбоната кальция с кислотами. Система позволяет лаборантам создавать заявки на расчет объема углекислого газа для реакций карбоната кальция (CaCO3), а преподавателям — управлять каталогом кислот и обрабатывать заявки. Также включены все дополнительные задания, полученные по ходу курса.
